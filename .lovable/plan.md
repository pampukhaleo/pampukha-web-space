## Цель

Перепозиционировать сайт: убрать слово «AI» (вызывает у клиентов ассоциацию «дёшево/некачественно»), вынести на первый план скорость и цену, снять страх качества блоком «процесс работы по шагам» и явной гарантией. Параллельно подключить Meta Pixel.

## 1. Новое позиционирование (везде, 3 языка: uk / en / pl)

Везде, где сейчас есть «AI / штучний інтелект / sztuczna inteligencja», заменяем на нейтральные формулировки про скорость, современность и удобство.

**Новый главный посыл (Hero):**
- UK: «Сучасні сайти під ключ — швидко і доступно. Готовий сайт за 3–5 днів, від $300»
- EN: «Modern websites done for you — fast and affordable. Live in 3–5 days, from $300»
- PL: «Nowoczesne strony pod klucz — szybko i niedrogo. Gotowe w 3–5 dni, od $300»

CTA остаются: «Безкоштовна консультація» / «Мої роботи».

## 3. Блок «Процесс работы по шагам» (новый)

Новый компонент `src/components/ProcessSection.tsx`, вставить в `src/pages/Index.tsx` между `AboutSection` и `PortfolioSection`.

1. **Бриф** — обсуждаем задачу, цели, конкурентов (бесплатно, 30 мин)

3. **Разработка и правки** — показываю прогресс, вносим правки вместе

Стиль: card-based, как существующие секции (`bg-card border border-border rounded-lg`), тёмная тема, акцент `text-primary`. Без градиентов/blur (по memory).

## 5. Facebook (Meta) Pixel через GTM

Pixel НЕ вшиваем в код. Подготавливаю инструкцию в чате:
1. В GTM создать новый тег **Custom HTML** со стандартным сниппетом `fbq('init', PIXEL_ID); fbq('track','PageView');`
2. Триггер — **All Pages**
3. Для событий формы создать второй тег с триггером по `dataLayer event: form_submit_success` (он уже шлётся из `ContactForm`, надо только проверить имя события в `docs/EVENTS_REFERENCE.md`)

В коде менять ничего не нужно — `dataLayer.push` уже работает (см. `src/components/SEO/Analytics.tsx`). От пользователя нужен только Pixel ID, либо он сам добавит тег в GTM по инструкции.

## Что НЕ делаем

- Не переименовываем файл `WhyAISPASection.tsx` и id `#why-ai-spa` — чтобы не сломать внутренние якори/аналитику. Меняем только видимый контент.
- Не трогаем дизайн-систему, цвета, шрифты.
- Не добавляем отзывы клиентов (пользователь выбрал процесс+гарантию, не отзывы).
- Не вшиваем fbq в `index.html` — идёт через GTM по выбору пользователя.
